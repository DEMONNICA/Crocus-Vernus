> [!IMPORTANT]
> Features ✨:
> 1. Kernel scheduler optimization — disables unnecessary logging, panic triggers, and improves scheduler responsiveness.
> 2. VM parameter tuning — swappiness, dirty ratios, overcommit, OOM behavior, and cache management.
> 3. Network core tuning — buffer sizes, queue discipline, backlog, and throughput parameters.
> 4. Intelligent I/O scheduler detection and adaptive block device optimization with RAM-based read-ahead tuning.
> 5. CPU governor tuning with performance-first approach for improved responsiveness.
> 6. Dynamic ZRAM optimization with algorithm selection and RAM-based sizing.
> 7. Automatic temporary file cleanup across system cache directories on every boot.
> 8. Sysctl backup — all modified values are saved on first write and restored automatically on uninstall.
